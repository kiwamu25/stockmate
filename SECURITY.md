# Security Policy

## Supported versions

This project currently supports security fixes for:

- `main` branch (latest code)

## Reporting a vulnerability

Please do not open a public issue for security vulnerabilities.

Preferred process:

1. Use GitHub Security Advisories (private report) for this repository.
2. Include details to reproduce the issue:
   - affected endpoint or page
   - impact
   - reproduction steps
   - proof of concept (if available)
3. Share potential mitigations if you have them.

We will acknowledge receipt and triage as quickly as possible.
